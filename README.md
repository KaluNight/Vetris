# Vetris
A new genre of Tetris game
## Controls:
- Up arrow to turn shape
- Left and right arrows to move
- Down arrow to fall faster
- Space to let the shape fall
- Shift to move faster
- Enter to use tiny shape
- P to pause
- C to change color
